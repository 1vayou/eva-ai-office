# Web Workspace Notes

## Purpose

The web workspace provides visibility into assistant branches, task progress, approvals, and history.

## Main pages

- Overview dashboard
- Assistant branch pages
- Approval queue
- Activity timeline

## UX rule

Telegram stays concise. Web stays deep.

## Branch model

The user sees one unified office, but can inspect specialist branches:
- Mail Chief
- Builder
- Daykeeper
- Signal
- Voice

Chief of Staff remains the summary layer and monitoring layer.
