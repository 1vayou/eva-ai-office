# Eva AI Office

Telegram + Web multi-assistant system for Eva's personal office.

## Vision

Eva AI Office is a private AI operations layer for founder life and work:
- one main Telegram assistant (`Chief of Staff`)
- multiple specialist assistants behind the scenes
- a web workspace for branch-level visibility
- approval queues for risky actions
- monitoring and orchestration by the Chief

## Core assistants

- Chief of Staff — routing, monitoring, summarization
- Mail Chief — inbox triage, drafts, action extraction
- Builder — product + engineering execution
- Daykeeper — schedule and meeting truth
- Signal — market and opportunity intelligence
- Voice — social media triage and drafting

## Product shape

- **Telegram**: primary conversation and quick approvals
- **Web workspace**: branch details, assistant status, approvals, activity timeline

## Initial prototype scope

- assistant registry
- routing rules
- approval policy
- prompt drafts
- static web workspace prototype
- implementation plan for Telegram + Web
